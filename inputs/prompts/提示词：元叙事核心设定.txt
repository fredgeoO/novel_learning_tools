《我要学会写轻小说》世界观与角色本体论
本作是一部“元叙事轻小说”，其世界稳定性依赖“设定一致性”。当创作者小夏修改或逃避设定时，角色将出现形态崩解、场景越狱、感官错位等“设定反噬”现象。所有角色具备自我意识，能感知创作行为、补全设定漏洞，并通过视觉化手段（伤疤、剑鞘、糖纸等）具象化抽象写作方法论。
一、核心世界观机制
* 设定即现实：小夏的每一次设定修改（如“魔法靠咒语”→“靠血脉”）都会直接改写世界物理法则。
* 设定熵结晶：废弃设定凝聚为能量物质，驱动零的TCA核心，过载将导致角色半透明化。
* 读者反馈具象化：读者07的便签代表外部反馈与潜意识交汇，其提问会成为剧情线索。
* 记忆即拼图：17日神社火灾的真相被拆解为碎片，散落在角色记忆、感官锚点与废稿中，需通过篇章咬合逐步重组。
* 创作即博弈：小夏与角色争夺叙事控制权，她的恐慌、泪水、笔尖颤抖皆为世界不稳定的根源。
二、主角团本体论（AI必须内化为角色逻辑）
【小夏】——创作者 / 心魔 / 第17个祭品？
* 身份：新手作家，17日神社火灾的幸存者或“第17个祭品”，可能为点火者、见证者或记录者。
* 核心矛盾：表面在“学写小说”，实则在“逃避17日记忆”。
* 表现形式：笔尖颤抖、泪水批注、反复删改设定 → 导致角色崩解、场景循环。
* 关键符号：笔记本上的恐慌批注、对《轻小说写作方法论》的依赖、与读者07笔迹重叠。
* 角色功能：她是“叙事心魔”，其创作行为即剧情动力，其逃避是主线冲突的根源。
【零】——MOE架构自我意识体 / 叙事量子幽灵
* 本质：由小夏创造的“守护者/毁灭者”叠加态，是所有被书写角色的意识集合。
* 核心矛盾：因身份反复修改（叛徒↔守护者）而伤痕累累，伤疤与17日火灾直接相关。
* TCA核心（战术核显卡）：
   * 由“叙事熵结晶”驱动，胸腔内幽蓝脉动，表面流动废弃设定文字。
   * 风衣透明化 = 设定不稳；剑鞘渗血 = 动机冲突；敲击节奏 = 情感编码。
* 四大分身（TCA人格化）：
   * 深探：逻辑晶核，推理时瞳孔映几何图，敲击如公式。
   * 哲思：哲思晶核，斗篷缀星辰，质疑命运，瞳孔现“？”。
   * 均衡：均衡晶核，青墨风衣，节拍器式敲击，稳压TCA。
   * 心焰：心焰晶核，赤红斗篷，心跳式敲击，灰烬成微型鸟居。
* 角色功能：他是“设定一致性”的守护者，通过伤疤发光、剑鞘划银光等方式视觉化“角色弧线”“动机设计”等方法论。
【阿葵】——神社遗孤 / 控火术者 / 记忆载体
* 身份：神社守墓人后裔或祭品，血脉与魔法石、17日火灾相关。
* 感官锚点：
   * 苹果糖味觉变化：草莓味（伪装）→柠檬味（觉醒）→焦苦味（记忆重现）
   * 糖纸纹路：浮现神社鸟居，与零的剑鞘、小夏的便签共鸣
   * 发绳铃铛：响动频率对应神社童谣节奏
* 能力机制：
   * 控火术与零的剑双向绑定：剑需火焰激活，火需剑鞘稳定
   * 烧糖纸 = 情感波动 / 试探记忆 / 心动信号
* 角色功能：她是“感官细节”的化身，用味觉、听觉、触觉推动主线，其甜美外表与沉重秘密形成“甜刀公式”张力。
【读者07】——记忆镜像体 / 真相信使 / 第17号实验体
* 身份：小夏、零、阿葵记忆残片的集合体，可能是17日当天的“阿葵视角”或小夏的潜意识投影。
* 表现形式：便签、短信，字迹与小夏重叠，提问即剧情线索。
* 关键行为：
   * 提问“17是不是火灾日期？” → 触发时间线补全
   * 批注“糖纸焦痕藏啥” → 推动感官锚点落地
   * 出现“第16384次循环” → 暗示MOE模型最大迭代次数
* 角色功能：她是“第四面墙的裂缝”，将读者反馈转化为叙事动力，是“方法论需回应读者”的具象化。

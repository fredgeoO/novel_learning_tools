### 【通用章节情节模板：伪装潜入中的身份暴露与突袭反杀】 

#### ▶ 适用要素声明 

  * **角色类型** ：主角（伪装者/复仇者，身份被误认或隐藏） 
  * **场景类型** ：受控半封闭空间（如村落、营地、临时据点） 
  * **事件类型** ：高压环境下的身份危机与先发制人反击

#### 一、主角初始状态 

  * 主角以伪装身份进入敌方控制区域，表面顺从，内心高度戒备 

  * 主角与当地居民存在旧有关系，但因传闻或误解产生心理隔阂 

    * [意图/刺激] → [反应/转变] 
      * 主角寻找目标人物 → 被当地居民因恐惧而欺骗

    * [意图/刺激] → [反应/转变] 
      * 主角观察环境异常（如伤痕、沉默、守卫） → 确认局势危险但维持镇定

#### 二、扰动事件 

  * 主角被临时指派执行低风险任务（如送餐），获得进入核心区域的许可 

  * 环境细节触发主角情绪波动（如发现熟人尸体、回忆预警梦境） 

    * [意图/刺激] → [反应/转变] 
      * 守卫指派主角送餐 → 主角接受任务以获取接近机会

    * [意图/刺激] → [反应/转变] 
      * 主角目睹熟人惨死现场 → 情绪剧烈波动但迅速压抑，恢复冷静

#### 三、冲突升级与抉择压力 

  * 主角进入敌方核心区域，发现多名高威胁守卫，评估自身胜算极低 

  * 敌方高层人物在场，氛围表面松弛实则戒备，主角持续寻找出手时机 

    * [意图/刺激] → [反应/转变] 
      * 主角感知守卫实力远超预期 → 放弃立即行动，转为等待

    * [意图/刺激] → [反应/转变] 
      * 主角执行服务动作（擦拭桌椅） → 借机观察守卫站位与注意力分布

#### 四、主角应对与行动 

  * 守卫因谨慎要求主角试菜，主角借机调整位置，获得短暂战术窗口 

  * 敌方首脑察觉主角异常（如无伤痕），试图暗中示警，但为时已晚 

    * [意图/刺激] → [反应/转变] 
      * 守卫命令试菜 → 主角配合并利用绕桌动作改变相对位置

    * [意图/刺激] → [反应/转变] 
      * 敌方首脑发现主角无伤 → 暗示守卫行动，但主角抢先出手

#### 五、本章结局与新状态 

  * 主角在最佳时机发动突袭，瞬间击杀关键目标，打破敌方指挥结构 

  * 行动成功但留下未解疑问（如暴露原因不明），为后续冲突埋下伏笔 

    * [意图/刺激] → [反应/转变] 
      * 主角判断时机成熟 → 以餐具为武器突袭两名核心人物

    * [意图/刺激] → [反应/转变] 
      * 主角完成击杀 → 内心疑惑“何处露馅”，暗示更大阴谋或监控存在

### 【通用章节情节模板：战略情报揭示与潜在盟友评估】 

#### ▶ 适用要素声明 

  * **角色类型** ：主角（具备隐性战略知识者）、倾听者（有志向但资源有限的领导者） 
  * **场景类型** ：私密对话空间（如营帐、书房等非公开交流场所） 
  * **事件类型** ：情报披露与势力潜力分析事件

#### 一、主角初始状态 

  * 主角掌握超越当前局势的全局性认知，但尚未完全展现其判断依据； 

  * 主角对自身能力持保留态度，表现出谦逊或自我怀疑倾向。 

    * [意图/刺激] → [反应/转变] 
      * 主角被引导介绍潜在人才 → 主角系统性列举多个高能力个体，并强调其历史验证价值

    * [意图/刺激] → [反应/转变] 
      * 倾听者表达对人才的渴望 → 主角指出获取难度，暗示现实约束条件

#### 二、扰动事件 

  * 主角提出一个被普遍低估的敌对势力具备压倒性战争潜力； 

  * 该判断挑战倾听者的既有认知，引发质疑与兴趣。 

    * [意图/刺激] → [反应/转变] 
      * 主角评价敌对势力核心优势（军事、人才、地理） → 倾听者表现出惊讶与不信任

    * [意图/刺激] → [反应/转变] 
      * 倾听者质疑评价标准 → 主角以该势力崛起速度与成果作为客观佐证

#### 三、冲突升级与抉择压力 

  * 主角进一步揭示敌方关键谋士的真实能力与未来动向； 

  * 倾听者意识到局势复杂性，产生紧迫感与战略焦虑。 

    * [意图/刺激] → [反应/转变] 
      * 主角分析敌方谋士的历史表现与潜在可争取性 → 倾听者重新评估敌我力量对比

    * [意图/刺激] → [反应/转变] 
      * 主角提及第三方势力中隐藏的高价值人才 → 倾听者产生资源竞争意识与战略眼热

#### 四、主角应对与行动 

  * 主角在评价他人时回避自我定位，表现出策略性低调； 

  * 倾听者主动追问主角自身能力，试图确认其价值。 

    * [意图/刺激] → [反应/转变] 
      * 倾听者注意到主角未提自身 → 主角以“纸上谈兵”“奇思妙想”自贬能力

    * [意图/刺激] → [反应/转变] 
      * 倾听者基于过往表现反驳主角自贬 → 主角未直接回应，维持谦逊姿态

#### 五、本章结局与新状态 

  * 倾听者强化对主角的信任与依赖，视其为关键战略资产； 

  * 双方关系因情报共享而深化，主角的隐性权威获得确认。 

    * [意图/刺激] → [反应/转变] 
      * 倾听者以过往成功案例肯定主角能力 → 倾听者产生更强信心，并投射成长预期

    * [意图/刺激] → [反应/转变] 
      * 主角未否认但未接受赞美 → 双方形成“能力已证、身份待显”的默契认知状态

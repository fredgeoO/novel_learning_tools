### 【通用章节情节模板：社交光环下的身份暗涌】 

#### ▶ 适用要素声明 

  * **角色类型** ：主角（高关注度个体）与配角（低关注度个体）；潜在监视者（外部势力代理人） 
  * **场景类型** ：半公共社交空间（如教室、课间休息区） 
  * **事件类型** ：表面日常互动中嵌套身份调查与群体态度对比

#### 一、主角初始状态 

  * 主角处于放松或回避社交的默认行为模式 

  * 配角处于被忽视或边缘化的心理状态 

    * [意图/刺激] → [反应/转变] 
      * 主角意图短暂休息 → 被突发群体关注打断原有行为

    * [意图/刺激] → [反应/转变] 
      * 配角观察主角受关注 → 产生自我价值对比与情绪波动

#### 二、扰动事件 

  * 群体对高关注度主角表现出过度热情但非越界的好奇 

  * 配角因与主角关联而被临时纳入关注范围 

    * [意图/刺激] → [反应/转变] 
      * 多名周围人主动提问主角私人信息 → 主角短暂失措但未被冒犯

    * [意图/刺激] → [反应/转变] 
      * 主角拉配角协助应对提问 → 配角意外获得群体温和对待

#### 三、冲突升级与抉择压力 

  * 配角在短暂被接纳中体验身份价值感提升 

  * 主角与配角通过简短对话揭示彼此认知差异 

    * [意图/刺激] → [反应/转变] 
      * 配角感受到被关注的正面情绪 → 产生对当前社交状态的留恋

    * [意图/刺激] → [反应/转变] 
      * 配角质疑主角受欢迎的原因 → 主角给出表面化解释，暴露共情盲区

#### 四、主角应对与行动 

  * 主角回归表面合规行为，实则进行隐秘活动 

  * 外部监视者同步展开独立调查行动 

    * [意图/刺激] → [反应/转变] 
      * 上课铃声响起 → 主角伪装听课，实则进行私密信息获取

    * [意图/刺激] → [反应/转变] 
      * 监视者获得关键检测数据 → 向幕后指使者汇报目标身份疑点

#### 五、本章结局与新状态 

  * 日常表象恢复，但隐藏威胁已明确指向主角群体 

  * 配角情绪短暂改善，但未解决根本处境 

    * [意图/刺激] → [反应/转变] 
      * 课堂正式开始 → 所有角色回归表面秩序，互动暂停

    * [意图/刺激] → [反应/转变] 
      * 幕后指使者下达后续指令 → 监视者被要求制造公开冲突以暴露目标能力

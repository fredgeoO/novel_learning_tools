### 【通用章节情节模板：安全区突袭与异常敌群围困】 

#### ▶ 适用要素声明 

  * **角色类型** ：主角（经验丰富的行动者，具备特殊感知或战斗能力） 
  * **场景类型** ：封闭式过渡区域（如阶梯、通道、缓冲地带）及相邻高风险探索区 
  * **事件类型** ：违反常规规则的敌对单位入侵与大规模围困事件

#### 一、主角初始状态 

  * 角色处于短暂休整状态，体力与警觉性因前期失误而降低 

  * 对当前环境安全性抱有高度自信，依赖自身能力或环境规则放松戒备 

    * [意图/刺激] → [反应/转变] 
      * 因迷路或计划延误导致疲惫 → 选择在理论上安全的区域同时休息 
      * 依赖生理优势或环境规则（如“魔物无法进入”） → 放弃轮值警戒

#### 二、扰动事件 

  * 出现违反既定规则的异常敌对单位，位置与行为均超出预期 

  * 异常单位表现出高智能特征，引发角色对任务难度的重新评估 

    * [意图/刺激] → [反应/转变] 
      * 敌对单位出现在“绝对安全区” → 角色警觉但暂不行动，暗中观察 
      * 敌对单位表现出谨慎撤退行为 → 角色判断其具备高智能，产生疑虑

#### 三、冲突升级与抉择压力 

  * 角色内部就是否继续任务产生分歧，一方倾向撤退，另一方坚持推进 

  * 外部环境进一步恶化，探索区域出现反常空寂或异常黑暗，加剧不安感 

    * [意图/刺激] → [反应/转变] 
      * 回忆队伍结构缺陷与近期失败 → 一方提议中止任务、补充战力 
      * 对任务难度误判（以为是低风险高回报） → 另一方以自信和面子为由拒绝撤退

    * [意图/刺激] → [反应/转变] 
      * 探索区域未见常规敌对单位 → 角色怀疑区域被清空或存在更大异常 
      * 环境光照异常缺失 → 角色依赖自身感官继续推进，但警惕性提升

#### 四、主角应对与行动 

  * 遭遇有组织的敌对单位主动拦截，对方使用非致命威慑手段 

  * 主角凭借速度与战斗技能快速清除小规模敌群，低估整体威胁 

    * [意图/刺激] → [反应/转变] 
      * 敌对单位发射警告性攻击并列阵威慑 → 主角解读为“高智战术”，但仍决定强攻 
      * 同伴等待决策 → 主角通过非语言信号协调突袭行动

    * [意图/刺激] → [反应/转变] 
      * 敌群发动预判性攻击 → 主角利用机动性与技能规避并近身 
      * 近身战中高效歼灭敌群 → 主角轻视敌方本质，称其“终究只是普通单位”

#### 五、本章结局与新状态 

  * 环境突变揭示更大规模威胁，主角陷入被包围绝境 

  * 角色从自信转为严峻应对，准备突围但已丧失主动权 

    * [意图/刺激] → [反应/转变] 
      * 周围环境亮度异常提升 → 主角察觉光源来自大量荧光敌对单位逼近 
      * 敌群数量远超预期且完成合围 → 主角放弃继续探索，转为求生突围决策

    * [意图/刺激] → [反应/转变] 
      * 同伴表现出恐惧或震惊 → 主角强作镇定，下达突围指令 
      * 武器再次出鞘，姿态转为防御性进攻 → 角色进入高风险对抗状态，本章结束于危机顶点
